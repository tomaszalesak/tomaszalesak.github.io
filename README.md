# tomaszalesak.github.io

Personal portfolio website for [Tomáš Zálešák](https://www.tomaszalesak.eu).

## Tech Stack

- [Next.js](https://nextjs.org/) (static export)
- [React](https://react.dev/)
- [Radix UI Themes](https://www.radix-ui.com/themes)
- [Tailwind CSS](https://tailwindcss.com/)
- [Bun](https://bun.sh/)

## Development

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
```

Static output is generated in the `out/` directory.

## Deployment

Deployed to [GitHub Pages](https://pages.github.com/) via GitHub Actions on push to `main`.
